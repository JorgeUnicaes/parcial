Sobre la materia

Gestion de Servidores Web es una materia que tiene el objetivo de que el alumno adquiera los conocimientos necesarios para la correcta instalación y configuración de Servidores Web con sistemas operativos propietarios y open source (licencia libre), y además configurar clúster de servidores para servicios de alta disponibilidad y despliegue de sitios y aplicaciones web..

Portafolio

En este apartado se encuentran las practicas desarrolladas a los largo del ciclo en la materia de Gestion de Servidores Web.

Primer Periodo

Practica 1 - Intalacion de VMWare

Objetivo de la practica: El objetivo de esta practica era qué el estudiante, realice los pasos necesarios para la instalación del software VMWare Workstation Pro, en la plataforma del sistema operativo Windows, para poder configurar máquinas virtuales que servirán como laboratorio de pruebas de Servidores Web..

Ver más...

Practica 2 - Instalacion de Windows Server 2019

Objetivo de la practica:En esta actividad el el objetivo era que alumno pudiera en un primer momento, descargar del sitio oficial Centro de Evaluación de Microsoft, una imagen ISO para instalar un Servidor Windows 2019, el cual es una de las herramientas esenciales para la configuración de un servidor web que más adelante se podrá poner en servicio.

Practica 3 - Instalación de Docker en Windows

Objetivo de la practica: Qué el estudiante, pueda realizar los pasos necesarios para la instalación y configuración de Contenedores Docker en ambiente Windows para implementar aplicaciones web.
